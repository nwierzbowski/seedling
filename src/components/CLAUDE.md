# UI Components - src/components/

This directory contains reusable React components for the Seedling frontend application.

## Components

- **AgentsTab.tsx**: Agent management interface with agent cards and switching functionality
- **TerminalTab.tsx**: Interactive terminal interface with Tauri integration
- **SettingsTab.tsx**: Configuration panel for system and agent settings

- **AgentsMenu.tsx**: Dropdown menu for agent selection
- **EngineerAgent.tsx**: Engineer agent pane component
- **ReviewerAgent.tsx**: Reviewer agent pane component
- **TesterAgent.tsx**: Tester agent pane component

## Patterns

- Tab-based navigation structure
- Tauri event system integration
- CSS modules for scoped styling
- Responsive grid layouts
